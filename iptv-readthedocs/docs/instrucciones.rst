Instrucciones rápidas
=====================

Copia cualquier enlace ``.m3u`` / ``.m3u8`` y pégalo en tu reproductor:

Smart TV (IPTV Smarters / TiviMate)
-----------------------------------

1. Abre **IPTV Smarters** o **TiviMate**.
2. Selecciona *Add playlist* / *Agregar lista*.
3. Pega el URL ``.m3u`` y guarda.

VLC (PC / Mac)
--------------

1. Abre VLC.
2. Menú: **Medio → Abrir ubicación de red...**
3. Pega el enlace ``.m3u`` y pulsa *Reproducir*.

Kodi
----

1. Instala el complemento **PVR IPTV Simple Client**.
2. Configúralo con la URL de la lista M3U.
3. Activa el cliente y reinicia Kodi.

.. note::

   Para mejor experiencia usa conexión estable. Respetamos el contenido
   libre · Solo fines informativos / recopilación de streams públicos.
